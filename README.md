Loading bar për arsye rekreacionale.

E implementume n'C; pa e përdor `string.h` header file.

Qysh me ekzekutu?

Në sistemin tuaj UNIX-based (Linux, MacOS, ...):
1. Klononi këtë repo:
```bash
git clone https://github.com/gentzhushi/loading-bar.git && cd loading-bar
```

3. Përpiloni kodin burimor:
```bash
cc loading_bar.c -o loading-bar
```

5. Ekzekutoni programin:
```bash
./loading-bar
```
