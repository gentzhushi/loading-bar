Loading bar për arsye rekreacionale.

E implementume n'C; pa e përdor `string.h` header file.

Qysh me ekzekutu?

Në sistemin tuaj UNIX-based (Linux, MacOS, ...):
1. Klononi këtë repo
`git clone https://github.com/gentzhushi/loading-bar.git && cd $_`

2. Përpiloni kodin burimor:
`cc loading_bar.c -o loading-bar`

3. Ekzekutoni programin:
`./loading-bar`
